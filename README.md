# 💰 Bank Loan Analysis Dashboard

## 📌 Project Overview

This project analyzes bank loan data to understand customer repayment behavior, credit policy impact, loan purposes, interest rate patterns, and customer risk profiles using Python and Tableau.

The project focuses on extracting meaningful business insights and building an interactive dashboard for loan analysis and KPI monitoring.

---

## 🎯 Objectives

* Analyze loan application trends
* Study repayment behavior
* Identify risky customer profiles
* Explore credit policy impact
* Monitor important KPI metrics
* Build an interactive Tableau dashboard
* Generate business insights from financial data

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Tableau

---

## 📂 Dataset Information

The dataset contains customer loan records including:

* Loan purpose
* Interest rate
* FICO score
* Installment amount
* Credit policy status
* Repayment status
* Revolving balance
* Debt-to-income ratio

---

## 📊 KPI Metrics

The dashboard tracks:

* Total Applications
* Average Interest Rate
* Average FICO Score
* Total Installment Amount

## 📌 Tableau Dashboard

The interactive Tableau dashboard provides:

* Loan purpose analysis
* Credit policy insights
* Repayment behavior analysis
* Customer risk evaluation
* Interest rate trends
* KPI monitoring

---

## 🔍 Key Insights

* Most customers follow the credit policy.
* Debt consolidation is the most common loan purpose.
* Higher FICO scores improve repayment performance.
* Customers with low credit scores show higher risk.
* Interest rates increase with risky customer profiles.
* Most loans are repaid successfully.
